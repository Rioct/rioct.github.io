Position

SmallPox has six differents positions:

Top leftTop centerTop right
Bottom leftBottom centerBottom right

In mobile (max-width:30em), all go down.

// top left example
spop({
	template: 'Position top left',
	position  : 'top-left',
	style: 'success'
});'

Autoclose

Autoclose, to... close... automatically... Woohoo never done before

Autoclose 
spop({
	template: '3 seconds autoclose',
	autoclose: 3000
});

Groups

There can only be one SmallPop from each group

GroupNo groupGroup 
spop({
	template: 'All fields are required!',
	group: 'submit-satus',
	style: 'error'
});

spop('Nothing here...');

spop({
	template: 'Your information has been submitted',
	group: 'submit-satus',
	style: 'success'
	autoclose: 2000
});

Callbacks

Do what you need onOpen and onClose

Callbacks 
spop({
	template: 'Please, close me.',
	style:'warning',
	onOpen: function () {
		document.body.style.background = "#fff";
	},
	onClose: function() {
		document.body.style.background = "";
		spop({
			template: 'Thank you!',
			style: 'success',
			autoclose: 2000
		});
	}
});

Event

In your template you can call the close event, just add data-spop="close"

Launch 
spop('Got to defaults');

Change Defaults

Change the default options of all SmallPop's

Changed all defaultsRestore defaults 
spop.defaults = {
	style     : 'error',
	autoclose : 5000,
	position  : 'top-left'
};

spop('Defaults changed! See the others examples.');

Options

spop({
	template  : null,// string required. Without it nothing happens!
	style     : 'info',// error or success
	autoclose : false,// miliseconds
	position  : 'top-right',// top-left top-center bottom-left bottom-center bottom-right
	icon      : true,// or false
	group     : false,// string, add a id reference 
	onOpen    : funtion() { },
	onClose   : funtion() { }
});
